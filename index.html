Library Scope

Warum die Oberflaeche so reduziert bleibt

Die Demo soll zeigen, wie weit ihr mit zwei kleinen Plain-JS-Libs kommt, ohne dass die UI die eigentliche Library-Idee ueberlagert.

  • Kein Framework, keine Bundler-Magie, keine Styling-Dependency.
  • Die QR-Matrix bleibt von der SVG-Aesthetik komplett getrennt.
  • Die Buehne rechts zeigt bewusst sofort das Resultat statt erst eine Tool-Karte.
